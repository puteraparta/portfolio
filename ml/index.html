Travel Insurance Claim Prediction Classification

Data Scientist Program - 2024
Github

This project focuses on helping a travel insurance company predict the likelihood of policyholders filing claims across various products, including annual coverage and trip cancellation plans, which are sold through agencies and online platforms. To support these efforts, a classification model will be built using algorithms such as Logistic Regression, Decision Tree, KNN, Random Forest, XGBoost, and LightGBM. Since the data is imbalanced, the model will ensure both classes are treated equally, with ROC/AUC and balanced accuracy as the primary evaluation metrics.
